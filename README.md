# 🧮 Calculadora com Tkinter  

Este é um projeto de **calculadora simples** desenvolvido em **Python** usando a biblioteca **Tkinter** para criar a interface gráfica.  


## 🚀 **Funcionalidades**  
✅ Operações básicas: soma, subtração, multiplicação e divisão.  
✅ Interface gráfica amigável feita com Tkinter.  
✅ Botões interativos e layout responsivo.  
✅ Tratamento de erros para evitar divisões por zero.  

---

 🛠️ **Tecnologias Utilizadas**  
- Python 🐍  
- Tkinter (Interface gráfica)
